방법론

ViT feature lifting + PE 제거

DINOv2를 4D로 끌어올리는 모듈과 positional embedding artifact를 제거하는 트릭을 봅니다.
오늘은 EmerNeRF의 두 번째 카드, vision foundation model feature를 4D로 끌어올리는 부분입니다. 동기는 단순합니다 — NeRF는 color·density는 잘하지만 semantic이 비어 있어 occupancy prediction 같은 perception에 못 씁니다. 그래서 DINOv2(ViT-B/14, stride 7, PCA로 768→64) 또는 DINOv1을 2D feature로 뽑은 뒤, feature head $f_* = MLP_{feat}(g_*)$ 와 feature sky head로 volume render해 학습합니다. 여기서 저자들이 발견한 비자명한 문제가 positional embedding pattern입니다. Transformer 기반 ViT는 PE 때문에 같은 텍스처라도 이미지 좌표에 따라 다른 feature를 내놓습니다. 그 결과 PE 패턴이 이미지에 고정된 채로 viewpoint와 무관하게 따라다녀 3D multi-view consistency를 정면으로 깨뜨립니다. Fig 1(h)에서 이 격자 무늬가 또렷이 보입니다. 해법은 우아합니다 — PE 패턴은 거의 모든 이미지에서 동일하므로 'shared learnable additive prior' 하나로 충분하다는 관찰. 학습 가능한 2D feature map $U$ (80×120×32)를 두고, 픽셀 좌표 $(u, v)$ 에서 bilinear interpolation 후 한 층 MLPPE로 PE feature를 디코드해, volume-render된 PE-free feature에 더해 ground truth와 매칭합니다: $\hat{F} = (\text{PE-free rendered}) + MLP_{PE}(\text{interp}(U, (u,v)))$. 이 단순한 additive decomposition으로 어떤 일이 일어나는가? Occupancy prediction에서 PE 제거만으로 DINOv2 micro-accuracy가 45.13%→62.05%, 평균 +37.50% relative improvement, 최대(Static-32) +63.22%. Feature-PSNR도 DINOv2에서 +0.83~+0.42 일관 향상(Table 4). 더 흥미로운 부분은 시각적으로 'PE에 안 영향받는 듯한' DINOv1도 PE 제거 후 +24.95% 평균 향상이 나옵니다 — 즉 눈에 안 보여도 PE는 항상 3D 표현을 갉아먹고 있었다는 얘기. 내일은 이 결과들의 숫자를 어떻게 자기 연구에 인용할지 봅니다.

Trigger Type B
Positional embedding (PE) artifact in ViT features
ViT는 패치 토큰에 위치 임베딩을 더해 attention을 계산하므로, 출력 feature에 이미지 좌표에 의존하는 잔류 패턴이 새깁니다. 이 패턴은 viewpoint가 바뀌어도 픽셀 좌표 기준으로 거의 동일하게 유지되어, NeRF가 의존하는 3D multi-view consistency를 깨뜨립니다. 결과적으로 PE를 그대로 4D로 lifting하면 동일한 3D 점이 시점마다 다른 semantic feature를 갖게 되어 occupancy classification·few-shot perception에서 정확도가 크게 떨어집니다. DINOv2가 시각적으로도 격자 무늬가 더 또렷하고 영향도 큽니다(최대 +63.22% 상대 향상).
Shared learnable additive prior $U$
PE 패턴이 이미지 사이에 거의 동일하다는 관찰에 기반해, 전역적으로 공유되는 학습 가능한 2D feature map $U \in \mathbb{R}^{80 \times 120 \times 32}$ 하나로 PE 성분을 흡수합니다. 각 픽셀에서 $U$ 를 bilinear interpolation해 단일 층 $MLP_{PE}$ 로 디코드한 결과를 volume-render된 PE-free feature에 더해 GT와 매칭. 핵심 가정은 'PE는 additive noise'이므로 분리 가능하다는 것. 학습이 끝나면 $U$ 만 빼서 PE-free feature를 얻고, 이게 3D 일관성 있는 semantic 표현이 됩니다. 단순하면서 ablation 없이도 작동 원리가 또렷한 트릭.

객관식 Q1
ViT PE 패턴이 4D feature lifting을 망치는 본질적 이유는?
같은 3D 점이 시점마다 픽셀 좌표가 달라지면서 다른 PE 성분을 받게 되므로, 4D 일관성을 직접 위반합니다.
객관식 Q2
Shared learnable additive prior $U$ 의 핵심 가정은?
Additive 가정이 식 11의 분해를 정당화하며, 단일 U로 PE를 흡수해도 충분히 작동함을 실험이 확인합니다.
주관식
본인이 쓰는 ViT 백본(DINOv2, SAM, EVA-CLIP 등)에서 PE artifact가 downstream 3D 태스크에 영향을 줄 가능성은 어디인가요? Occupancy·BEV detection·NeRF distillation 중 어느 자리가 가장 위험할까요?
힌트
Per-image grid pattern이 viewpoint와 무관하게 따라다니는지 한 장면을 다른 시점에서 PCA 시각화해 보세요.
꼭 짚고 가야 할 것
Foundation model을 3D로 lifting할 때, 2D에서는 보이지 않던 PE 잔류가 거의 항상 multi-view 일관성을 갉아먹습니다. 'PE-free'는 옵션이 아니라 기본기여야 합니다.
