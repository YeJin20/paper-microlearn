오늘의 학습 · DAY 4
개요

Landscape에서의 좌표

동적 NeRF·AV NeRF·scene flow 계열을 비교축 위에 놓고 EmerNeRF의 빈자리를 명확히 합니다.
오늘은 EmerNeRF를 지도 위에 좌표화합니다. 비교축은 세 개로 잡으면 깔끔합니다 — (1) 동적 객체를 모델링하는가, (2) 외부 라벨/사전학습 모델에 의존하는가, (3) 단일 traversal에서 동작하는가. 이걸로 다섯 그룹을 보겠습니다. HyperNeRF/Nerfies: 동적 O, 라벨 X, 단일 traversal O이지만 canonical deformation 가정으로 운전 enter/exit에 실패. D²NeRF: 동적 O, 라벨 X, monocular 가정. 도시 야외에서 분해 실패(Dynamic-32에서 24.35 PSNR). NSG·UniSim: 동적 O, GT 박스/트래킹 의존, 단일 traversal O. 라벨 비용에서 막힘. SUDS: 동적 O, pre-trained flow 의존, multi-traversal 가정. flow warping 비용 큼. StreetSurf·iNGP: 정적만, 라벨 X, 단일 traversal O — 정적 최고치(26.15 PSNR)이지만 동적은 손도 못 댐. NSFP: scene flow only, per-timestep optimization으로 매우 느림. EmerNeRF의 좌표는 '동적 O + 라벨 X + 사전학습 flow X + 단일 traversal O'라는 마지막 빈 칸입니다. 그 결과 Dynamic-32에서 +3.70 PSNR, Static-32에서 +2.93 PSNR, EPE3D 0.014m(NSFP 0.365m 대비)로 모든 축에서 SOTA를 가져갑니다. 핵심은 '한 가지를 잘하는 방법은 많지만, 네 조건을 동시에 만족하는 방법이 없었다'는 빈자리에 자기지도 + flow emergence라는 두 카드로 끼워 넣은 점입니다. 내일은 이 카드들이 어떻게 한 파이프라인으로 묶이는지 큰 그림을 펴봅니다.

Trigger Type A
Annotation-free dynamic decomposition
정-동 분해를 GT bounding box나 사전학습 segmentation 없이 수행하는 능력입니다. NSG/UniSim이 라벨로 분해를 우회한 데 비해, EmerNeRF·D²NeRF·SUDS는 라벨 없이 분해를 시도합니다. 하지만 D²NeRF는 도시 야외에서 hyperparameter 민감도로, SUDS는 pre-trained flow 의존으로 한계가 분명합니다. EmerNeRF는 density regularization $\mathcal{L}_{\sigma_d} = E(\sigma_d)$ 라는 단순한 prior 하나로 이 자리를 메웠다는 점이 핵심입니다.
NOTR benchmark
Waymo Open Dataset에서 손으로 큐레이팅한 120개 시퀀스 벤치마크로, Static-32 / Dynamic-32 / Diverse-56 세 split으로 나뉩니다. Diverse-56은 ego-static, high-speed, exposure mismatch, dusk/dawn, gloomy, rainy, night 7개 카테고리로 구성되어 도시 운전 환경의 imbalance를 정면으로 노출시킵니다. 또한 2D dynamic bounding box, GT 3D scene flow, 3D semantic occupancy까지 함께 제공해 reconstruction 외에도 motion·semantic 평가가 가능하다는 점이 차별점입니다.

객관식 Q1
다음 중 EmerNeRF가 유일하게 동시에 만족하는 조건 조합은?
다른 조합은 NSG·UniSim(1), SUDS(2), StreetSurf(4)가 이미 점유. EmerNeRF는 셋 모두를 라벨 없이 푸는 빈자리를 차지했습니다.
객관식 Q2
NOTR가 기존 AV 데이터셋과 가장 다른 점은?
기존 Waymo·nuScenes는 단순 시퀀스가 다수라 동적·악조건 평가를 가리지 못합니다. NOTR는 의도적으로 균형을 맞춘 120 시퀀스 큐레이션입니다.
주관식
본인 팀에서 운영 중인 운전 데이터셋의 imbalance는 어디에 있나요? Diverse-56의 7개 카테고리 중 가장 부족한 것은 무엇이고, 그것을 NeRF로 합성 보강할 때 어떤 신호가 가장 먼저 깨질까요?
힌트
야간·exposure mismatch는 색 일관성을, 고속·ego-static은 flow 학습 신호를 비대칭적으로 흔듭니다.
꼭 짚고 가야 할 것
Benchmark imbalance는 모델 평가의 평균에 가려져 거의 항상 corner case 실패를 숨깁니다. Diverse split을 따로 들고 다니는 습관이 곧 안전 마진입니다.
