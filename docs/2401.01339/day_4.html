개요

분리 표현의 네 갈래, Street Gaussians는 어디?

NSG·MARS·EmerNeRF·deformation 4D-GS와의 trade-off 비교, Street Gaussians의 좌표 짚기
같은 '동적 거리 모델링' 문제를 푼 친구들을 좌표 위에 놓아봅시다. 두 축은 (A) 차량을 어떻게 분리하느냐, (B) 표현이 implicit이냐 explicit이냐. • NSG: latent code per-object + MPI 배경. 차량은 ID-conditioned NeRF로 디코딩. 분리는 분명하지만 background가 MPI라 시점 자유도가 좁고 렌더는 0.47 FPS. • MARS: Nerfstudio 위에 scene graph 구성. tracker box로 차량을 canonical하게 정렬한 뒤 NeRF로 학습. 품질은 NSG보다 낫지만 raymarching 비용은 그대로(0.68 FPS). • EmerNeRF: tracker box 없이도 self-supervision으로 static/dynamic field를 분해. annotation-free라는 장점이 큰 대신 instance 단위로 쪼개주지 않아 '차 한 대 빼기' 같은 편집이 본질적으로 어렵습니다. • Deformation 기반 4D-GS: 가우시안에 시간 deformation을 입혀 동적 표현. 소규모 객체엔 강력하지만 도시 거리처럼 차량이 수십 미터 이동하는 대변위에선 motion이 너무 크고 instance 분리도 모호. Street Gaussians의 좌표는 'explicit + instance-aware + tracker-pose 결합'. 머릿속에 4분면을 그려놓고, 각 방법이 어느 칸에 들어가는지 한번 직접 배치해 보세요. 어디가 비어있는지가 이 논문의 빈 자리입니다.
