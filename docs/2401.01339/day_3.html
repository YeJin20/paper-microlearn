오늘의 학습 · DAY 3
개요

한 문장으로 보는 핵심 아이디어

한 문장 요약 + 직관 비유 + 왜 이전보다 나은가
오늘은 핵심을 한 문장으로 정리해봅시다. 한 문장 요약: Street Gaussians는 정적 배경과 움직이는 차량을 각각 3D Gaussian 점군으로 분리 모델링하고, 추적된 차량 pose와 4D Spherical Harmonics로 시간 변화 외형까지 명시적으로 표현하는 기법입니다. 직관/비유: 도시를 거대한 디오라마라고 상상해보세요. 배경 점토는 그대로 두고, 차량 미니어처들만 따로 떼어 손에 들고 돌려보거나 교체할 수 있는 구조입니다. NeRF가 모든 걸 하나의 검정 상자에 욱여넣은 거라면, 이쪽은 블록을 분리해 조립하는 LEGO 같은 셈이죠. 왜 더 나은가: 점군 기반 explicit representation이라 학습이 30분, 렌더링이 135 FPS까지 가속됩니다. 게다가 객체와 배경이 따로 정의되어 있어 회전·이동·교체 같은 편집이 raymarching 없이 자연스럽게 가능해요. 즉 '품질·속도·편집성'의 삼각구도를 한꺼번에 푸는 카드를 explicit Gaussian이라는 표현 선택으로 손에 쥔 셈입니다.

Trigger Type C
3D Gaussian Splatting (3D GS)
장면을 anisotropic 3D Gaussian의 집합으로 표현하고 화면에 splatting해 렌더링하는 explicit 방법. NeRF보다 훨씬 빠르다.
Explicit Scene Representation
장면을 점·메쉬·가우시안처럼 직접 다룰 수 있는 기하 단위로 저장하는 표현. implicit MLP와 달리 편집·조작이 용이하다.
4D Spherical Harmonics
시간 차원을 추가한 SH 표현. 각 SH 계수를 시간의 푸리에 함수로 두어 동일 점이 시간에 따라 다른 색을 내도록 한다.

객관식 Q1
Street Gaussians의 핵심 한 문장 요약으로 가장 정확한 것은?
장면 분해 + 점군 기반 3D Gaussian + 4D SH 외형 모델이라는 세 축이 한 문장 요약의 본질이다.
객관식 Q2
이 접근이 NeRF 합성 표현보다 우월한 본질적 이유는?
표현이 explicit이라는 점이 속도와 편집성을 동시에 잡는 결정적 차이다.
주관식
'블록을 분리해 조립하는 LEGO' 비유는 어디까지 통하고 어디서부터 깨질까?
힌트
객체 간 상호작용 — 그림자, 반사, occlusion을 떠올려보세요.
꼭 짚고 가야 할 것
객체 분리는 기하 편집엔 깔끔하지만, 객체와 배경 사이의 빛 상호작용(shadow/reflection)은 4D SH 같은 별도 장치가 없으면 LEGO 비유가 무너진다.
