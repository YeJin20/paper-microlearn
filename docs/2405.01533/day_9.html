결과 — DriveLM에서 +3점, ego-status 없이 collision 3.22% → 1.90%

페이퍼에 명시된 핵심 수치만 인용하여 OmniDrive 데이터가 실제로 무엇을 끌어올렸는지 자율주행 관점에서 정리.
오늘은 숫자 정리. 단, 페이퍼에 명시된 것만 인용합니다. DriveLM Q&A 벤치마크 (Tab. 3, Omni-L 기준) - DriveLM만 사전학습: 평균 score 0.53 - + OmniDrive 사전학습 추가: 0.56 - + LLaVA665k 추가: 0.57 - 둘 다 결합: 0.58 → OmniDrive 자동 생성 데이터만으로 +3pt. 자동 데이터인데도 품질이 받쳐줍니다. nuScenes Open-loop planning, ego-status 없이 (Tab. 2, Omni-L) - trajectory-only 학습 (Omni-L‡): collision 3.22%, intersection 3.90% - OmniDrive Q&A 함께 학습 (Omni-L): collision 1.90%, intersection 3.29% → counterfactual reasoning이 plan distribution을 살린다는 직접 증거. Counterfactual reasoning quality (Tab. 4) — Omni-L Safe P/R = 72.1/58.0, Omni-Q = 70.7/49.0. 페이퍼 결론은 'VLM 출신 백본이 평균 우세, 단 Collision detection은 Omni-Q가 강함'. 자율주행 관점의 의미는 두 가지 — 자동 데이터로 만든 reasoning supervision이 단순 행동 모방을 보완하고, ego-status 의존을 낮춥니다. 내일은 마지막 — 이걸 scene editing 기반 data augmentation과 어떻게 잇는지로 마무리합니다.
